# 🚀 AutoSend Letters HeadHunter

## 📝 Описание проекта

**AutoSend Letters HeadHunter** — это продвинутый скрипт, который революционизирует поиск работы на HeadHunter благодаря автоматизации отправки откликов и **инновационной AI-генерации персонализированных сопроводительных писем**. Теперь каждый отклик будет уникальным и персонализированным под конкретную вакансию!

## 🎯 Зачем нужен этот скрипт?

Поиск работы — это трудоемкий процесс: просмотр сотен вакансий, составление индивидуальных сопроводительных писем, отслеживание откликов. В итоге удается отправить лишь несколько качественных откликов в день.

**AutoSend Letters HeadHunter** автоматизирует рутинные задачи и использует силу искусственного интеллекта для создания персонализированных писем, освобождая время для подготовки к собеседованиям и развития профессиональных навыков.

## 🚀 Быстрая установка

### 👥 Для пользователей (рекомендуется)

1. **Установите расширение Scripty** для Chrome:
   - [Scripty Extension](https://chrome.google.com/webstore/detail/scripty-javascript-inject/milkbiaeapddfnpenedfgbfdacpbcbam) 📥

2. **Установите скрипт автоматически**:
   - [Установить скрипт одним кликом](https://scripty.abhisheksatre.com/#/share/script_mbb7h4qpayicsv) ⚡

3. **Настройте через удобный интерфейс**:
   - Откройте [hh.ru](https://hh.ru) 🌐
   - Нажмите **"⚙️ Настройки AutoSend"** в верхней навигации
   - Следуйте пошаговым инструкциям настройки

### 👨‍💻 Для разработчиков

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/1MaxSpb/autosend-letters-hh.git
   cd autosend-letters-hh
   ```

2. **Установите зависимости**:
   ```bash
   npm install
   ```

3. **Соберите проект**:
   ```bash
   npm run build
   ```

4. **Используйте готовый файл**:
   - Скопируйте содержимое `dist/script.js`
   - Вставьте в консоль браузера на HeadHunter

## ⚙️ Настройка AI-генерации

### 🎯 Поддерживаемые нейросети

| Провайдер | Качество | Стоимость | Скорость | Рекомендуется для |
|-----------|----------|-----------|----------|------------------|
| **ChatGPT** | ⭐⭐⭐⭐⭐ | 💰💰 | ⚡⚡⚡ | Максимального качества |
| **Yandex GPT** | ⭐⭐⭐⭐ | 💰 | ⚡⚡ | Русскоязычных вакансий |
| **GigaChat** | ⭐⭐⭐ | 💰 | ⚡⚡ | Бюджетного варианта |
| **Gemini** | ⭐⭐⭐⭐ | 💰 | ⚡⚡⚡ | Быстрой генерации |

### 🔧 Пошаговая настройка

1. **Получите API ключ** у выбранного провайдера
2. **Откройте настройки** на HeadHunter (кнопка "⚙️ Настройки AutoSend")
3. **Включите AI генерацию** и выберите провайдера
4. **Введите API ключ** и сохраните настройки
5. **Готово!** Теперь письма будут генерироваться автоматически

## 🛠️ Устранение проблем

### ❓ Частые вопросы

**Q: AI не генерирует письма**  
A: Проверьте API ключ и баланс аккаунта у провайдера

**Q: Скрипт не находит резюме**  
A: Убедитесь, что ID резюме указан правильно в настройках

**Q: Отклики отправляются слишком быстро**  
A: Увеличьте задержку в настройках до 2000-3000 мс

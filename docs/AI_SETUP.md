# 🤖 Настройка AI для генерации сопроводительных писем

AutoSend Letters HeadHunter теперь поддерживает генерацию персонализированных сопроводительных писем с помощью нейросетей. Это значительно повышает шансы получить приглашение на собеседование.

## 🎯 Поддерживаемые AI-провайдеры

### 1. ChatGPT (OpenAI)
- **Модель**: GPT-3.5-turbo
- **Качество**: Отличное
- **Стоимость**: Платная (от $0.002 за 1000 токенов)
- **Скорость**: Быстрая

### 2. Yandex GPT
- **Модель**: YandexGPT Lite
- **Качество**: Очень хорошее для русского языка
- **Стоимость**: Есть бесплатный лимит
- **Скорость**: Средняя

### 3. GigaChat (Сбер)
- **Модель**: GigaChat
- **Качество**: Хорошее для русского языка
- **Стоимость**: Есть бесплатный тариф
- **Скорость**: Средняя

### 4. Google Gemini
- **Модель**: Gemini Pro
- **Качество**: Отличное
- **Стоимость**: Есть бесплатный лимит
- **Скорость**: Быстрая

## 🔧 Пошаговая настройка

### Шаг 1: Получение API ключей

#### OpenAI (ChatGPT)
1. Перейдите на [platform.openai.com](https://platform.openai.com/api-keys)
2. Войдите в аккаунт или зарегистрируйтесь
3. Нажмите "Create new secret key"
4. Скопируйте полученный ключ (начинается с `sk-`)
5. **Важно**: Пополните баланс для использования API

#### Yandex GPT
1. Перейдите в [Yandex Cloud Console](https://console.cloud.yandex.ru/)
2. Создайте новый проект или выберите существующий
3. Перейдите в раздел "Сервисные аккаунты"
4. Создайте новый сервисный аккаунт
5. Назначьте роль `ai.languageModels.user`
6. Создайте API-ключ для сервисного аккаунта
7. Скопируйте Folder ID проекта (находится в настройках проекта)

#### GigaChat (Сбер)
1. Перейдите на [developers.sber.ru](https://developers.sber.ru/portal/products/gigachat)
2. Зарегистрируйтесь и подтвердите аккаунт
3. Создайте новое приложение
4. Получите токен авторизации
5. Скопируйте токен

#### Google Gemini
1. Перейдите в [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Войдите с Google аккаунтом
3. Нажмите "Create API Key" 
4. Выберите проект или создайте новый
5. Скопируйте созданный API ключ

### Шаг 2: Настройка в скрипте

1. **Откройте HeadHunter** в браузере
2. **Нажмите кнопку "⚙️ Настройки AutoSend"** в верхней навигации
3. **Включите AI генерацию** - поставьте галочку "Включить AI генерацию сопроводительных писем"
4. **Выберите провайдера** из выпадающего списка
5. **Введите API ключ** в соответствующее поле
6. **Для Yandex**: также введите Folder ID
7. **Нажмите "💾 Сохранить настройки"**

### Шаг 3: Проверка работы

1. Найдите любую вакансию на HeadHunter
2. Нажмите "Откликнуться"
3. Если AI настроен корректно, письмо будет сгенерировано автоматически
4. В консоли браузера (F12) будет сообщение "Генерируем сопроводительное письмо с помощью AI..."

## ⚙️ Дополнительные настройки

### Изменение промпта для AI

Если вы хотите изменить стиль генерируемых писем, отредактируйте файл `src/config/aiConfig.js`:

```javascript
systemPrompt: `Ваш кастомный промпт здесь...
Название вакансии: {vacancyName}
Создай сопроводительное письмо:`
```

### Настройка параметров генерации

В том же файле можно изменить:

```javascript
generation: {
  maxTokens: 200,     // Максимальное количество токенов
  temperature: 0.7,   // Креативность (0.0 - 1.0)
  timeout: 10000      // Таймаут запроса (мс)
}
```

## 🛠️ Устранение проблем

### AI не работает
1. **Проверьте API ключ** - он должен быть действительным
2. **Проверьте баланс** аккаунта у провайдера
3. **Откройте консоль браузера** (F12) для просмотра ошибок
4. **Проверьте интернет-соединение**

### Письма генерируются слишком долго
1. **Увеличьте таймаут** в настройках
2. **Попробуйте другого провайдера**
3. **Проверьте загрузку сети провайдера**

### Письма не соответствуют ожиданиям
1. **Измените промпт** для лучшего результата
2. **Попробуйте другую температуру** генерации
3. **Используйте другого провайдера**
